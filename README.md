# 🌍 TripNest – Smart Travel Planner

> Plan your journeys effortlessly. Discover destinations. Build unforgettable experiences.

---

## 🚀 Overview

TripNest is a full-stack travel planning application designed to simplify how people explore, organize, and manage trips. Whether you're planning a weekend getaway or a long vacation, TripNest provides a seamless experience to handle everything in one place.

---

## ✨ Features

- 🌎 Discover popular travel destinations
- 🗺️ Plan and manage trips efficiently
- 📅 Create personalized itineraries
- 🔐 Secure authentication system
- 💾 Save and revisit your favorite places
- 📱 Responsive UI for all devices
- ⚡ Fast and smooth user experience

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS / CSS

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**Other Tools**
- Git & GitHub
- REST APIs

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository
```bash
git clone https://github.com/vedantdubey19/TripNest.git
cd TripNest
2️⃣ Install dependencies
npm install

3️⃣ Setup environment variables
MONGO_URI=your_mongodb_uri
PORT=5000
JWT_SECRET=your_secret_key
4️⃣ Run the project
npm run dev

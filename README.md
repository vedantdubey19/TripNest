# TripNest 🏠

TripNest is a premium, full-stack travel destination listing and review platform. It allows users to browse unique stays, leave detailed reviews with ratings, and view exact locations on an interactive map.

## ✨ Features
- **Listing Management**: Users can create, edit, and delete their own travel listings.
- **Image Uploads**: Integration with Cloudinary for seamless high-quality image storage.
- **Interactive Maps**: Maps powered by Mapbox for visualizing listing locations.
- **User Authentication**: Secure signup and login system using Passport.js.
- **Responsive Design**: Modern, glassmorphism-inspired UI that works beautifully on all devices.
- **Review System**: Detailed user reviews with star ratings powered by `starability`.

## 🛠️ Tech Stack
- **Frontend**: EJS, Bootstrap 5, Vanilla CSS
- **Backend**: Node.js, Express.js
- **Database**: MongoDB (Mongoose ODM)
- **Session Management**: express-session with connect-mongo
- **Authentication**: Passport.js

## 🚀 Getting Started
1. Clone the repository.
2. Install dependencies: `npm install`
3. Configure your `.env` file with `MAP_TOKEN` and Cloudinary keys.
4. Start the server: `npm start`
5. Visit `http://localhost:8080` in your browser.
